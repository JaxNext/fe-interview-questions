# Q

下列说法正确的是？

1. `display: none;` 不为被隐藏的对象保留其物理空间；
2. `visibility: hidden;` 所占据的空间位置仍然存在，仅为视觉上的完全透明；
3. `visibility: hidden;` 产生 reflow 和 repaint (回流与重绘)；
4. `visibility: hidden;` 与 `display: none;` 两者没有本质上的区别；

# A

1，2
