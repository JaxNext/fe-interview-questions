# Q

给定字符串，返回无元音字母的版本。

# A



# 总结



<script>
  function func(str) {
    return str.replaceAll(/[aeiou]/gi, '')
  }
  console.log(func('hello'))
  console.log(func('world'))
  console.log(func('hello world'))
  console.log(func('aeiou'))
  console.log(func('AEIOU'))
  
  
</script>
