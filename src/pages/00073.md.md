# 题干

下面哪些方式在同一个窗口下能够检测一个js对象是数组类型？（ ）

1. Array.isArray()
2. instanceof
3. typeof
4. Object.prototype.toString.call()

# 正确答案

1、2、4

# 总结

- [] instanceof Array
- typeof array 返回 Object
- toString 返回 [Object Array]
