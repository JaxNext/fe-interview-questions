# 题干

以下结果里，返回 `false` 的是？

1. [] == true
2. !![]
3. NaN == NaN
4. null == undefined

# 正确答案

1、3

# 总结

- [] == true 中，[] 转为 0，true 转为 1。
    - 两侧类型不同，尝试转为数字
    - 右侧转为数字是 1
    - 左侧先调用 valueOf()，返回数组本身；再调用 toString() 返回空字符串
    - 空字符串转数字是 0
- NaN 与任何值都不相等。
- [] 转为 Boolean 为 true。
    - ! 取反会把运算值转为布尔；数组转布尔为 true
    - 两次取反得到 true
- null == undefined
