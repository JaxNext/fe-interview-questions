# 题干

这段代码的运行结果是什么？

```jsx
(function() {
    var x=foo();
    var foo=function foo() {
        return "foobar"
    };
    return x;
})();
```

# 正确答案

`Uncaught TypeError: foo is not a function`

# 总结

- `var foo` 会被提升顶端
- 执行 `var x = foo()` 时，foo 的值为 undefined。
