# Q

- 给定一个数字，将其各个数位上的数字相乘，直到得到一位数，
- 返回相乘的次数。

# A



# 总结



<script>
  function func(n, count = 0) {
    if (n < 10) {
      return count
    }
    const m = n.toString().split('').map(Number).reduce((acc, cur) => acc * cur, 1)
    return func(m, count + 1)
  }
  console.log(func(88))
</script>
