# 题干

- 给定一个数字，作为宝塔的层数，大于 0；
- 返回一个由字符串组成的数组；
- 用星号代表瓦。

# 总结



<script>
  function func(n) {
    for (let i = 0; i < n; i++) {
      const space = ' '.repeat(n - i - 1)
      const middle = '*'.repeat(2 * i + 1)
      console.log(`${space}${middle}${space}`)
    }
  }

  console.log(func(9))
