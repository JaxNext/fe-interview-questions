# Q

给定一个字符串，将其中的字母替换为器在字母表中的序号（从 1 开始），非英文字母忽略。例子：

```bash
'abc' => '1 2 3'
```

# 答案

```javascript
function formatter(str) {
    if (!str) return ''
    return str.toLowerCase()
    .match(/[a-z]/g)
    .map(char => char.charCodeAt() - 'a'.charCodeAt() + 1)
    .join(' ')
}
```

# 总结

- 用 str.match(/[a-z]/g) 获取字符串中所有小写字母
