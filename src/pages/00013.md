# Q

- 给定一个可遍历结构；
- 返回一个数组；
- 数组元素为输入值去重后按原顺序排列；
- 区分大小写。

# 总结

<script>
    function formatter(input) {
        const res = []
        const map = new Map()
        for (const x of input) {
            map.set(x, (map.get(x) || 0) + 1)
            if (map.get(x) === 1) {
                res.push(x)
            }
        }
        return res
    }
    console.log(formatter('abccba'))
    console.log(formatter('abccbaa'))
    console.log(formatter('abccbaab'))
</script>
