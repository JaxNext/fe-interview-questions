问题：[] ==![] 的结果是什么？

答案：true

原理：

1. 先算右侧
    1. ![] 会把数组转为 Boolean，然后取反
    2. false、0、null、undefined、NaN 这几个转为布尔是 false，其他都是 true
    3. 所以 [] 变成了 true，取反为 false
    4. 此时表达式为 [] == false，两侧类型不同，要转为相同类型（数字）
    5. 所以右侧被转为数字 0
2. 再算左侧
    1. [] 为对象类型，要先转为原始类型
    2. 数组调用 valueOf 的结果还是数组
    3. 所以接着调用 toString() 得到 ‘’
    4. 空字符串转数字得到 0
3. 0 == 0
